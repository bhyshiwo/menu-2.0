# 自定义字体设置说明与使用案例

## 字体体系架构

小程序端共有 **三层字体 + 两个独立字体**，互不冲突、按优先级叠加：

```
全局基础字体（--font-family）
  └─ 来源：UI主题默认字体 → 自定义全局字体覆盖（customFont）
  └─ 影响范围：页面所有未特别指定的文字

艺术字体（--artistic-font）
  └─ 来源：customFont（不填 → inherit，回退到全局字体）
  └─ 影响范围：标语、菜品名称
  └─ 优先级高于全局字体

标题字体（--title-font）
  └─ 来源：titleFont（不填 → inherit，回退到全局/艺术字体）
  └─ 影响范围：小程序名称、分类标题、购物车标题、弹窗标题、订单详情标题、成功页标题

Google Fonts
  └─ 来源：UI主题自带 + 艺术字体/标题字体各自的加载URL
  └─ 动态加载 <link> 标签，避免重复加载
```

---

## 一、管理端设置入口

所有字体设置位于：**后台管理 → 小程序设置** 页面，共三个独立的字体区块：

### 1.1 🎨 自定义字体与形状（全局覆盖）

| 字段 | 说明 |
|------|------|
| **自定义字体** | CSS `font-family` 值，会**完全覆盖**当前主题的默认字体 |
| 留空 | 使用主题默认字体（如「宫崎骏」主题的 Noto Serif SC） |

**影响范围**：页面全局 `--font-family` 变量，影响所有未单独设字体的元素。

> 示例：填入 `'ZCOOL QingKe HuangYou', cursive` → 全局变活泼手写风

---

### 1.2 🖼️ 艺术字体（标语 & 菜品名）

| 字段 | 说明 |
|------|------|
| **CSS font-family** | 只作用于**标语**和**菜品名称** |
| **Google Font 加载地址** | 该字体在 Google Fonts 的 URL（用于加载字体文件） |

**影响范围**（仅这两个 CSS 类）：
- `.app-slogan` — 小程序标语
- `.dish-name` — 菜品列表中的菜品名

> 不填 → 回退到全局 `--font-family`

---

### 1.3 📝 标题字体（所有标题）

| 字段 | 说明 |
|------|------|
| **CSS font-family** | 只作用于所有 **标题类文字** |
| **Google Font 加载地址** | 该字体在 Google Fonts 的 URL |

**影响范围**：
- `.app-name` — 小程序名称（这个元素同时应用了 title-font + artistic-font → title-font 优先）
- `.category-title` — 左侧分类栏标题
- `.cart-detail-title` — 购物车详情标题
- `.checkout-header h2` — 结算弹窗标题
- `.success-modal h2` — 下单成功弹窗标题
- `.order-detail-items-title` — 订单详情标题

> 不填 → 回退到 `--artistic-font` → 再回退到 `--font-family`

## 二、Google Fonts 使用指南

### 2.1 获取字体 URL

访问 [fonts.google.com](https://fonts.google.com)，选择中文字体（勾选 "Chinese (Simplified)"），记录两部分信息：

| 需要记录 | 用途 | 示例 |
|----------|------|------|
| CSS 规则里的 `font-family` | 填到「CSS font-family」输入框 | `'Ma Shan Zheng', cursive` |
| `<link>` 标签里的 `href` | 填到「Google Font 加载地址」输入框 | `https://fonts.googleapis.com/css2?family=Ma+Shan+Zheng&display=swap` |

### 2.2 可脱离 Google Fonts 使用系统自带字体

不填 Google Font 加载地址也可用，只要 `font-family` 中指定的字体在用户设备上已安装，或使用系统回退字体：

| 平台 | 可用楷体 | font-family 写法 |
|------|----------|-------------------|
| Windows | 楷体 | `'KaiTi', 'STKaiti', serif` |
| macOS | 华文楷体 | `'STKaiti', 'KaiTi', serif` |
| iOS | 苹方 | `'PingFang SC', sans-serif` |
| Android | 思源黑体 | `'Noto Sans SC', sans-serif` |

## 四、完整使用案例

### 案例 1：国风餐厅（标题用书法、正文用楷体）

**操作步骤**：

1. 全局字体（自定义字体与形状 → 自定义字体）：
   ```
   'Noto Serif SC', 'STKaiti', 'KaiTi', serif
   ```

2. 艺术字体（标语 & 菜品名）：
   - CSS font-family：`'Ma Shan Zheng', cursive`
   - Google Font：`https://fonts.googleapis.com/css2?family=Ma+Shan+Zheng&display=swap`

3. 标题字体（标题）：
   - CSS font-family：`'ZCOOL QingKe HuangYou', cursive`
   - Google Font：`https://fonts.googleapis.com/css2?family=ZCOOL+QingKe+HuangYou&display=swap`

**效果**：页面基础用楷体，标语和菜名用毛笔手写，标题用飘逸行书。

---

### 案例 2：日式居酒屋（简洁明快）

**操作步骤**：

1. 全局字体（自定义字体与形状 → 自定义字体）：
   ```
   'Noto Serif SC', 'Hiragino Mincho ProN', serif
   ```

2. 艺术字体：留空（使用全局字体）

3. 标题字体：
   - CSS font-family：`'ZCOOL XiaoWei', serif`
   - Google Font：`https://fonts.googleapis.com/css2?family=ZCOOL+XiaoWei&display=swap`

**效果**：整体素雅，标题有小薇体的精致感。

---

### 案例 3：奶茶甜品店（可爱活泼）

**操作步骤**：

1. 全局字体（自定义字体与形状 → 自定义字体）：
   ```
   'ZCOOL KuaiLe', 'PingFang SC', sans-serif
   ```
   Google Font 不需要（主题「动漫」已自带加载）

2. 艺术字体：
   - CSS font-family：`'Long Cang', 'ZCOOL KuaiLe', cursive`
   - Google Font：`https://fonts.googleapis.com/css2?family=Long+Cang&display=swap`

3. 标题字体：
   - CSS font-family：`'ZCOOL KuaiLe', sans-serif`
   - 无需 URL（主题已加载）

**效果**：全局活泼，菜品名更飘逸，标题保持快勒体的统一感。

---

### 案例 4：不依赖 Google Fonts，纯系统字体方案

适用于离线环境或不想加载外部资源：

| 字段 | 填写内容 |
|------|----------|
| 全局自定义字体 | `'STKaiti', 'KaiTi', '楷体', serif` |
| 艺术字体 CSS | `'KaiTi', '楷体', serif` |
| 艺术字体 URL | 留空 |
| 标题字体 CSS | `'STKaiti', 'KaiTi', serif` |
| 标题字体 URL | 留空 |

## 五、常见问题

### Q：只填 font-family 不填 Google Font URL 可以吗？

可以。只要用户设备已安装该字体就能正常显示；未安装则回退到 fallback 字体（即 font-family 中靠后的备选字体）。

### Q：多个字体设置之间会冲突吗？

不会。系统按严格优先级链工作：
```
标题文字 → --title-font → --artistic-font → --font-family
```

### Q：如何确认字体是否生效？

1. 小程序端按 F12 → 检查元素 → Computed → 查看 `font-family`
2. 如果显示的是你的自定义字体名，说明加载成功
3. 如果显示 fallback 字体（如 PingFang SC），检查 Google Font URL 是否正确、网络是否通畅

### Q：字体加载慢怎么办？

- 只选一个字体族（避免加载太多）
- 用系统自带字体替代 Google Fonts（完全不需要网络）
- Google Fonts URL 末尾加 `&display=swap`（已默认添加），确保文字先显示再替换字体

| 你想控制 | 设置位置 | CSS 变量 | 覆盖元素 |
|----------|----------|----------|----------|
| 页面全部文字的默认字体 | 自定义字体与形状 → 自定义字体 | `--font-family` | 所有元素 |

| 字体文件加载 | 艺术字体 → URL / 标题字体 → URL | — | 加载到浏览器 |
